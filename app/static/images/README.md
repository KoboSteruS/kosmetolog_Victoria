# Папка изображений

Поместите сюда следующие изображения:

## Обязательные файлы:

1. **logo.png** - Логотип клиники (темный, для светлого фона)
2. **logo-light.png** - Светлый логотип (для темного фона Hero секции)
3. **favicon.ico** - Иконка для браузера
4. **hero-bg.jpg** - Фоновое изображение для первого экрана (рекомендуемый размер: 1920x1080px)

## Изображения клиники:

5. **clinic1.jpg** - Интерьер клиники / зона ожидания
6. **clinic2.jpg** - Кабинет косметолога
7. **clinic3.jpg** - Процедурный кабинет

## Фотографии специалистов:

8. **doctor1.jpg** - Елена Васильева (Главный врач)
9. **doctor2.jpg** - Виктория Коваленко (Врач-косметолог)
10. **doctor3.jpg** - Ксения Михайлова (Врач-косметолог)
11. **doctor4.jpg** - Ольга Андреева (Врач-косметолог)

## Рекомендации:

- Формат: JPG для фото, PNG для логотипов
- Оптимизируйте изображения для web (сжатие)
- Рекомендуемые размеры:
  - Логотипы: 300x100px (прозрачный фон)
  - Hero фон: 1920x1080px
  - Интерьер: 800x600px
  - Специалисты: 400x500px (портретная ориентация)
  
## Placeholder изображения:

Если у вас нет готовых изображений, вы можете использовать:
- [Unsplash](https://unsplash.com) - бесплатные фото
- [Pexels](https://pexels.com) - бесплатные фото
- [Lorem Picsum](https://picsum.photos) - placeholder изображения

Пример использования placeholder:
```html
<img src="https://picsum.photos/800/600" alt="Clinic">
```
